A simple and elegant YouTube video downloader built with Python and PyQt5.

## Features

- **Clean GUI**: Modern, dark-themed interface with custom fonts and smooth animations
- **Quality Selection**: Choose from multiple video qualities (720p, 480p, 360p, 240p, 144p)
- **Easy to Use**: Just paste the YouTube URL, select quality, and download
- **Smart URL Handling**: Automatically cleans and validates YouTube URLs (supports both `youtube.com` and `youtu.be` formats)
- **Progressive Downloads**: Downloads videos with both audio and video combined

## Tech Stack

- **Python 3.x**
- **PyQt5**: For the graphical user interface
- **pytubefix**: For downloading YouTube videos
